# Java Chatting Application

A simple Java desktop chat application using sockets and Swing.

## Features

- Multiple clients connect to a single server
- Real-time messaging
- Swing-based GUI

## How to Run

### 1. Compile
```bash
javac src/*.java
```

### 2. Start the Server
```bash
java -cp src Server
```

### 3. Start Clients
```bash
java -cp src Client
```
